아이돌 팬덤 플랫폼 - JISOO

글로벌 유명 가수를 위한 모바일 팬덤 & 굿즈 플랫폼

Capacitor와 React Native를 활용하여 개발된 글로벌 팬덤 플랫폼입니다. 1차 버전은 Capacitor와 Native 모듈을 연동하여 굿즈 판매 및 NFC 음원 스트리밍 기능을 구현하였으며, 2차 버전은 React Native와 AWS EKS를 기반으로 대규모 동시 접속을 지원하는 글로벌 실시간 팬미팅 스트리밍, 구독 모델, 고도화된 쇼핑몰 기능을 제공합니다.

프로젝트 개요

기간: JISOO 1차 및 2차 운영 및 고도화 진행 중
역할: 모바일 앱 및 백엔드 개발 (Full Stack)
목적: 글로벌 팬덤을 위한 올인원 소통 및 커머스 플랫폼 구축

기술 스택

React Native Capacitor Nest.JS AWS EKS Native Modules NFC Streaming

주요 기능

  • 글로벌 실시간 팬미팅: 대규모 동시 접속자를 처리하는 라이브 스트리밍 시스템
  • 팬덤 커머스: 굿즈 판매 및 글로벌 결제 시스템 연동
  • NFC 음원 연동: 실물 앨범/굿즈와 연동되는 디지털 음원 스트리밍 (Native 모듈 활용)
  • 구독 시스템: 멤버십 전용 콘텐츠 및 혜택 제공
  • 크로스 플랫폼 지원: Android 및 iOS 동시 지원 (Capacitor -> React Native 고도화)

개발 성과

  • 성공적인 앱 스토어 출시 및 운영 (Android/iOS)
  • 대규모 트래픽 처리를 위한 AWS EKS 인프라 구축
  • Native 모듈 직접 개발을 통한 하드웨어 기능(NFC) 완벽 제어
  • 결제 및 구독 모델 도입으로 수익화 모델 구축
