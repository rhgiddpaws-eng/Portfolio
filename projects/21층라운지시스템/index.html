🎯 Digital Signage System

21층 라운지 디지털 사이니지 시스템

Unity 기반 호텔 디지털 사이니지 및 통합 관리 시스템

대형 시설의 1층 키즈존과 21층 라운지에 시청각 컨텐츠를 제공하는 시스템을 Unity로 개발했습니다. 아이들을 위한 애니메이션 컨텐츠부터 호텔 분위기를 고조시키는 시각적 컨텐츠, 날씨 정보 시스템까지 포함한 종합 디지털 사이니지 솔루션입니다. 또한 호텔 리뉴얼과 관련된 전반적인 관리, 전시, 광고, 홈페이지 시스템 개발에도 참여했습니다.

프로젝트 개요

클라이언트: 비공개
역할: Unity 기반 디지털 사이니지 시스템 개발
범위: 1층 키즈존 & 21층 라운지 대형 스크린 시스템
추가: 호텔 통합 관리/전시/광고/홈페이지 시스템

기술 스택

Unity C# Digital Signage Remote Control CMS Weather API Video Streaming

핵심 기능

  • 1층 키즈존: 어린이 애니메이션 컨텐츠 시스템
  • 21층 라운지: 분위기 연출 시각 컨텐츠
  • 날씨 시스템: 일주일 단위 날씨 정보 제공
  • 통합 관리: 호텔 전반 시스템 관리

주요 시스템 구성

🎬 1층 키즈존 시스템

어린이 고객을 위한 엔터테인먼트 공간에 최적화된 컨텐츠 제공 시스템

  • 애니메이션 컨텐츠 재생
  • 대형 스크린 최적화 UI/UX
  • 자동 재생 및 스케줄링
  • 밝고 친근한 인터페이스
  • 원격 컨텐츠 관리

🌆 21층 라운지 시스템

프리미엄 라운지 분위기를 고조시키는 시각적 컨텐츠 시스템

  • 고급스러운 시각 컨텐츠
  • 일주일 단위 날씨 정보 표시
  • 시간대별 분위기 연출
  • 실시간 정보 업데이트
  • 우아한 UI 디자인

🎛️ 통합 관리 시스템

호텔 리뉴얼에 따른 전반적인 디지털 시스템 관리

  • 컨텐츠 관리 시스템 (CMS)
  • 원격 제어 및 모니터링
  • 전시 및 광고 시스템
  • 홈페이지 연동
  • 통합 대시보드

핵심 성과 및 기술적 특징

🎮

Unity 기반 개발

Unity 엔진을 활용하여 고품질 시각 컨텐츠를 안정적으로 재생하는 디지털 사이니지 시스템을 구축했습니다. 운영 안전성과 뛰어난 그래픽 성능을 동시에 달성했습니다.

📺

대형 스크린 최적화

1층과 21층의 대형 스크린에 최적화된 UI/UX를 설계하여 멀리서도 명확하게 보이는 인터페이스를 구현했습니다. 고해상도 컨텐츠를 부드럽게 재생합니다.

🌤️

실시간 날씨 정보

21층 라운지에 일주일 단위 날씨 정보를 실시간으로 제공하여 고객의 편의성을 높였습니다. 기상 API 연동을 통한 정확한 정보 제공이 가능합니다.

🎯

맞춤형 컨텐츠

키즈존과 라운지의 서로 다른 타겟 고객층에 맞춘 차별화된 컨텐츠를 제공합니다. 각 공간의 특성과 분위기에 최적화된 시청각 경험을 구현했습니다.

🔧

원격 관리 시스템

컨텐츠 업데이트, 스케줄 관리, 시스템 모니터링을 원격으로 수행할 수 있는 통합 관리 시스템을 구축했습니다. 효율적인 운영이 가능합니다.

🏢

호텔 통합 솔루션

디지털 사이니지뿐만 아니라 호텔 리뉴얼에 따른 관리, 전시, 광고, 홈페이지 시스템까지 포함한 종합적인 디지털 솔루션을 제공했습니다.
