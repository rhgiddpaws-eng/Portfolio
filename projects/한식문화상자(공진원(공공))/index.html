한식문화상자 (HANSIK-DORAK)

공진원 공공기관 웹사이트 리뉴얼 프로젝트

한식 문화를 세계에 알리는 공공기관 웹사이트입니다. Spring Framework 기반의 풀스택 개발로 한식 레시피, 문화 체험 키트, 다국어 지원 등 다양한 콘텐츠를 제공하며, 전통 한식 문화를 현대적이고 접근하기 쉬운 방식으로 전달합니다.

프로젝트 개요

기간: 2025년 (공공기관 프로젝트)
역할: 풀스택 개발 (Backend & Frontend)
목적: 한식 문화 세계화 및 온라인 다국어 플랫폼 구현

기술 스택

Spring Framework JSP MySQL Apache Tomcat Bootstrap Maven Spring Security i18n (다국어)

주요 기능

  • 한식 레시피 데이터베이스 (100+ 레시피 등록 및 관리)
  • 문화 체험 키트 소개 및 온라인 신청 시스템
  • 다국어 지원 (한국어/영어) - i18n 적용
  • 반응형 웹 디자인 (PC, 태블릿, 모바일 최적화)
  • 검색 및 필터링 기능 (재료, 조리시간, 난이도별)
  • 관리자 페이지 (콘텐츠 관리, 사용자 관리)
  • 전통 문화 콘텐츠 통합 (한복, 전통 회화 등)
  • SEO 최적화 및 웹 접근성 준수

개발 성과

  • 공공기관 웹사이트 리뉴얼 성공적 완료
  • Spring MVC 패턴 기반 안정적인 백엔드 아키텍처 구축
  • 다국어 지원으로 글로벌 접근성 향상
  • 사용자 친화적 UI/UX 설계 및 구현
  • 안정적인 운영 및 유지보수 체계 확립
