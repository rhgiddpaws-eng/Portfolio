동남아시아 시장을 타겟으로 한 대규모 카지노 포털 서비스

한게임 카지노 수준의 퀄리티와 규모를 갖춘 모바일 카지노 플랫폼입니다. Unity 엔진을 기반으로 Android, iOS, Web 멀티 플랫폼을 지원하며, Java Spring 기반의 안정적인 서버 아키텍처로 대규모 동시 접속자를 원활하게 처리합니다.

▶ 프로젝트 시연 영상 보기 (YouTube)

PROJECT OVERVIEW

목표: 동남아 시장 점유율 확대를 위한 프리미엄 카지노 포털 구축
범위: 모바일 앱(Client) 및 게임 서버(Server) 전체 개발
특징: 대규모 트래픽 처리 인프라 및 크로스 플랫폼 지원

TECH STACK

Unity 3D Java Spring Android / iOS WebGL MyBatis Jenkins WebSocket

KEY HIGHLIGHTS

  • 멀티 플랫폼 지원
    Unity One-Source Multi-Use 전략으로 AOS, iOS, Web 환경 완벽 대응
  • 대규모 트래픽 처리
    동시 접속자(CCU) 수만 명 처리를 위한 최적화된 서버 아키텍처 및 DB 설계
  • CI/CD 자동화
    Jenkins를 활용한 빌드 및 배포 자동화 파이프라인 구축
  • 글로벌 서비스 최적화
    국가별 네트워크 환경을 고려한 패킷 최적화 및 다국어 시스템

PROJECT GALLERY

메인 로비 (Lobby)

인게임 화면 (In-Game)

몰입감 높은 게임 플레이 화면 및 이펙트 연출

이벤트 및 보상 (Events)

사용자 리텐션을 위한 데일리 이벤트 및 잭팟 보상 시스템
