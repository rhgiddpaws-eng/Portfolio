AlioSoft 문서변환 및 에디팅 솔루션

Enterprise Document Conversion & Web Editing Platform

기업 내 다양한 포맷의 문서를 웹 표준 형식으로 변환하고, 별도의 설치 없이 웹 브라우저에서 바로 열람 및 편집할 수 있는 통합 솔루션입니다. 대용량 문서의 고속 처리와 원본 수준의 높은 변환 품질을 보장하며, 안전한 협업 환경을 제공합니다.

PROJECT OVERVIEW

기간: 2022년 - 2023년
역할: 솔루션 아키텍처 설계 및 코어 모듈 개발 (Backend & Conversion Engine)
성과: 공공기관 및 대기업 납품, 기존 솔루션 대비 변환 속도 30% 향상

TECH STACK

Java Spring Boot React.js C++ / JNI LibreOffice SDK PDFium Redis Docker

KEY FEATURES

  • 범용 포맷 지원 HWP, DOCX, XLSX, PPTX, PDF 등 50여 종 이상의 다양한 문서 포맷 양방향 변환 지원
  • AOS / IOS native 지원 Web 뿐만 아니라 AOS / IOS native 앱에서도 문서 변환 및 편집 기능 제공
  • 웹 에디터 (Non-ActiveX) ActiveX 설치 없이 브라우저 내에서 HWP 및 MS Office 문서 편집 및 저장 기능 제공
  • 대용량 병렬 처리 비동기 큐 시스템(RabbitMQ/Redis) 기반의 대용량 문서 일괄 변환 및 부하 분산 처리
  • 보안 및 권한 관리 문서 열람/편집 권한 제어, 워터마크 삽입, DRM 연동을 통한 기업 정보 유출 방지
