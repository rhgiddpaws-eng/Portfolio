생성형 AI 숏츠 편집기

Vue3/Nuxt3 기반 웹 비디오 편집 플랫폼 - AI 생성 & FFmpeg 인코딩

생성형 AI를 활용한 웹 기반 숏츠 비디오 편집 플랫폼입니다. Stable Diffusion LoRA 파인튜닝으로 커스텀 이미지를 생성하고, Replicate API로 동영상을 생성합니다. Vue3/Nuxt3(TypeScript, Pinia) 프론트엔드, Spring Boot 백엔드, FastAPI AI 서비스가 Kafka로 비동기 통신하며, WebSocket으로 생성 진행 상태를 실시간 반영합니다. SSR/SSG 렌더링과 FFmpeg 타임라인 인코딩으로 MP4/HLS를 제공하고, Docker 환경에서 MariaDB, Kafka, Zookeeper를 운영하며 S3에 자산을 저장하는 확장 가능한 아키텍처입니다.

시스템 아키텍처 & 워크플로우

1

프론트엔드 (Vue3/Nuxt3)

사용자가 비디오 편집 인터페이스에서 AI 생성 및 편집을 요청. Pinia로 편집 상태를 관리하고 Spring Boot API와 REST로 통신하며, WebSocket으로 생성 진행률을 실시간 수신. SSR/SSG로 초기 로딩과 SEO를 최적화.

2

백엔드 (Spring Boot)

생성 요청을 Kafka 토픽(ai-generation-requests)으로 발행. 결과는 KafkaAIConsumer로 수신하여 MariaDB에 저장. 크레딧/결제 로직 관리.

3

AI 서비스 (FastAPI)

Kafka를 구독하여 Stable Diffusion LoRA 또는 Replicate API로 이미지/영상 생성. 결과를 Kafka 토픽(ai-generation-results)에 게시.

4

이미지 파인튜닝 (LoRA)

10-50장의 이미지와 캡션으로 Kohya_ss/diffusers를 사용해 UNet attention layer에 LoRA 어댑터 학습. Rank 32-128, 100+ epoch로 커스텀 스타일 생성.

5

비디오 인코딩 (FFmpeg)

Export API 호출 시 타임라인/레이어를 FFmpeg filter_complex로 변환. overlay/concat/xfade/subtitles 적용하여 H.264/AAC MP4 생성.

6

저장 & 스트리밍 (S3/HLS)

인코딩된 MP4를 S3에 업로드 (users/{userId}/assets/{category}/{assetId}/encoded.mp4). HLS master.m3u8 생성으로 스트리밍 제공.

7

인프라 (Docker)

MariaDB, Kafka, Zookeeper를 Docker Compose로 구성. FastAPI → Spring Boot → Nuxt3 순으로 서비스 기동. 수평 확장 가능.

핵심 성과 및 기술적 특징

🎨

Stable Diffusion LoRA 파인튜닝

스타크래프트 게임 캐릭터/스타일을 학습시킨 커스텀 이미지 생성 모델 개발. Kohya_ss와 diffusers 라이브러리로 UNet의 attention layer에 LoRA 어댑터를 학습시켜 독특한 스타일의 이미지를 생성합니다.

🎬

Replicate API 영상 생성

다양한 AI 모델을 옵션별로 테스트하며 최적의 영상 생성 파이프라인을 구축했습니다. 서버 상태를 모니터링하며 안정적인 영상 생성 서비스를 제공합니다.

⚡

Kafka 비동기 처리

Kafka를 활용한 이벤트 기반 아키텍처로 AI 생성 지연을 격리하고 시스템 안정성을 확보했습니다. FastAPI 인스턴스를 수평 확장하여 대규모 요청을 효율적으로 처리합니다.

🎞️

FFmpeg 고급 인코딩

filter_complex를 활용한 overlay, concat, xfade, subtitles 효과 적용. 타임라인 기반 편집 데이터를 H.264/AAC MP4로 변환하여 고품질 비디오를 생성합니다.

📺

HLS 스트리밍

MP4 인코딩과 함께 HLS (master.m3u8) 형식으로 변환하여 적응형 비트레이트 스트리밍을 제공합니다. 다양한 네트워크 환경에서 최적의 재생 경험을 보장합니다.

💳

크레딧 시스템

Spring Boot에서 크레딧 및 결제 로직을 관리합니다. 생성 실패 시 자동 환불, 성공 시 자산 저장으로 안정적인 과금 시스템을 구현했습니다.

🔒

인증된 다운로드

사용자별 컨텐츠 접근 제어를 위한 인증된 다운로드 엔드포인트 구현. S3 URL은 인증된 프록시를 통해서만 접근 가능하여 보안성을 강화했습니다.

🐳

Docker 컨테이너화

MariaDB, Kafka, Zookeeper를 Docker Compose로 구성하여 일관된 개발/배포 환경을 제공합니다. 서비스 간 의존성을 명확히 하여 안정적인 운영이 가능합니다.
