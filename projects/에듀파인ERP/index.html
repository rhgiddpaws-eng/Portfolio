에듀파인 ERP 웹 연동 시스템

아이큐브(iCube) 클론 기반으로 K-에듀파인 송수신 규격에 맞게 완벽하게 연동되는 강력한 웹 기반의 ERP 대시보드 및 데이터 인프라 시스템

Java Spring Boot React / Next.js iCube Clone K-에듀파인 연동 대시보드

유치원 재무·회계 시스템

예산의 책정부터 결산까지 한 눈에 파악할 수 있는 예산/결산 관리 시스템입니다.

예산과 결산

복잡한 예산 코드 체계(세입/세출)를 직관적인 트리 구조와 표 형태로 구성하여 예산 편성부터 지출 내역, 잔액을 실시간으로 관리할 수 있습니다.

이전 버전 ERP인 아이큐브(iCube)의 복잡한 매뉴얼을 최신 웹 환경에 맞춰 UI 편의성을 대폭 향상했습니다.

유치원 회계 / 현금출납

날짜별 수입, 지출 내역, 은행 계좌 잔고를 대조하여 빈틈없는 회계 관리가 가능합니다. 현금출납부 작성 시 계정과목 자동 매핑 및 필터링 검색을 지원합니다.

K-에듀파인 원클릭 데이터 전송

ERP에 입력된 결산, 계좌, 수입/지출 내역을 국가 K-에듀파인 시스템 전문 규격에 맞춰 자동 파싱하고 전송하는 중계 모듈이 탑재되어 있습니다.

인사 및 기본 자원 통합 관리

사용자 및 권한 관리

웹 기반에서 간편하게 유치원별 원장, 행정 실무자, 교사 등 세분화된 권한과 계정을 일괄 관리합니다.

인사 / 급여 관리

기본급, 수당, 4대보험 공제 등 급여 대장을 생성하고 연말정산 서식 등과 연동되는 복합적인 인사 모듈입니다.

실시간 계좌 연동

시중 은행 시스템과 오픈 API로 연계해, 거래내역 및 잔액 내역을 ERP에서 직접 조회할 수 있습니다.

기초 환경 설정

유치원 관리(단설/병설/사립) 설정, 사업 연도, 회계 기간 수립 및 각종 코드 데이터/키워드를 자유롭게 구성합니다.
